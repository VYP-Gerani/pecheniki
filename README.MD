# TaskMaster

**TaskMaster** — AI-бизнес задачник для контроля задач сотрудников, мотивации через баллы и лидерборд, а также анализа эффективности работы с помощью AI.  

Проект позволяет:
- Отслеживать выполнение задач и прогресс сотрудников.
- Стимулировать лучших сотрудников через систему баллов и наград.
- Анализировать эффективность работы и мотивацию команды.
- Обеспечивать безопасность данных и процессов.
